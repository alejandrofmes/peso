# Web-Based Employment Platform (Academic Project)
> **A  web-based employment platform designed to provide scalable
and centralized platforn for PESO Offices to aid in managing
applicant and employer data. It included employer job posting,
jobseeker application and recommendation, profile building,
training management, and reports generation and system and data
management.**

To achieve its goals, the system includes the following core functionalities:

* **Job Posting Management:** A dedicated module for posting vacancies, and upon PESO's approval, enabling jobseekers to view listings and submit applications.
* **User Profile & Resume Builder:** A module allowing applicants to build professional profiles and generate digital resumes within the platform.
* **Skills-Training Portal:** Features for posting skill-based trainings and courses endorsed by PESO, which includes participant registration.
* **Job-Matching:** An automated feature for providing job recommendations to users and assisting PESO staff in screening top-tier applicant recommendations.
* **Analytics & Reporting:** A module that generates pertinent reports for data-driven decision-making and employment tracking of PESO Admins.


## ðŸš€ Modules
* **Job Posting Module** A module that allows employers to create a job posting. Upon PESO admin's approval, this job posting would be available for jobseekers.
* **Job Listing Module** Allows jobseekers to search for, filter, view approved job postings, and get job vacancy recommendations based from their profile input. Upon viewing, jobseekers would be able to send their application.
* **Profile Module** This module allows jobseekers to build and update their profiles. They are also capable of uploading their resume, and generating resume bsed from their profile inputs. Employers are also capable of maintaining their profile inputs which can be viewed by users.
* **Training Module** This module allows PESO admins to post trainings or courses available for jobseekers, manage registrants, and track attendees. Jobseekers would be able to view and register to these trainings.
* **Reports Generation Module** Specific to PESO officer accounts, which allows them to manage, view, and track employment related data within the system.
